### Static helpers — story wrappers only

Fixed values that do not adapt to theme. Use only in Storybook wrappers and explicit overrides — never in component code.

| Tailwind class | Hex | Use |
|---|---|---|
| `bg-cream` | `#fffbf5` | Light page background |
| `bg-cream-surface` | `#f7f3f5` | Light surface background |
| `bg-dark-bg` | `#1c1a16` | Dark page background |
| `bg-dark-surface` | `#252220` | Dark surface background |

## Typography

| Role | Family | Tailwind class |
|---|---|---|
| Display / Headings | Cabinet Grotesk | `font-display` |
| Body | Areal | `font-body` |
| Editorial accent | Ogg | `font-editorial` |

**Sizes used:**

| Level | Size | Classes |
|---|---|---|
| H1 | 72px | `font-display font-bold tracking-tightest leading-none` |
| H2 | 48px | `font-display font-bold tracking-tighter leading-tight` |
| H3 | 32px | `font-display font-bold tracking-tighter leading-snug` |
| H4 | 20px | `font-display font-bold leading-snug` |
| Body large | 18px | `font-body leading-relaxed` |
| Body | 16px | `font-body leading-relaxed` |
| Body small | 14px | `font-body leading-relaxed` |

**Letter spacing:**
- `tracking-tightest` → `-0.04em` — hero headlines
- `tracking-tighter` → `-0.03em` — section titles

**Color:** All heading levels and body text use `text-fg`. One token, all sizes.

### Label pattern

Section eyebrows, form labels, and small metadata use a consistent mono label style — not one of the three main families, but used consistently throughout.

```
text-xs font-mono tracking-widest uppercase
```

- Section eyebrows: `color: var(--accent)`
- Form labels and metadata: `color: var(--fg)` at `opacity: 0.5`

## Spacing

All tokens are available as CSS custom properties and as Tailwind spacing utilities. Use whichever fits the context — values are identical.

> **Note:** The custom Tailwind keys (`gap-space-lg` etc.) only generate CSS when the class appears in a scanned file. When in doubt, use the standard numeric scale — `gap-6` and `gap-space-lg` are the same value.

| Token | px | CSS variable | Custom Tailwind | Standard Tailwind |
|---|---|---|---|---|
| xs  | 4px   | `--space-xs`  | `*-space-xs`  | `*-1`  |
| sm  | 8px   | `--space-sm`  | `*-space-sm`  | `*-2`  |
| md  | 16px  | `--space-md`  | `*-space-md`  | `*-4`  |
| lg  | 24px  | `--space-lg`  | `*-space-lg`  | `*-6`  |
| xl  | 32px  | `--space-xl`  | `*-space-xl`  | `*-8`  |
| 2xl | 48px  | `--space-2xl` | `*-space-2xl` | `*-12` |
| 3xl | 64px  | `--space-3xl` | `*-space-3xl` | `*-16` |
| 4xl | 96px  | `--space-4xl` | `*-space-4xl` | `*-24` |
| 5xl | 128px | `--space-5xl` | `*-space-5xl` | `*-32` |

Replace `*` with any spacing prefix — `p`, `gap`, `m`, `mt`, `px`, etc.
