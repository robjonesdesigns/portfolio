# Marquee

Horizontally scrolling ticker pinned to the bottom of the Hero section. Accepts an array of strings and scrolls them in a seamless loop using a CSS `@keyframes` animation.

The list is duplicated internally — pass the original set once, and the component handles the loop.

## Usage

```jsx
// In Hero.jsx
const skills = [
  'Product Design', 'UX Research', 'Design Systems',
  'Prototyping', 'Developer Handoff',
]

<Marquee items={skills} speed={30} />
```
