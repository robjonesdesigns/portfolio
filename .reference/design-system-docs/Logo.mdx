# Logo

The RJ mark is the primary brand identifier for the portfolio. It uses adaptive color tokens so it updates automatically with the active theme — no separate logo-specific values.

## Minimum size

<Canvas of={LogoStories.MinimumSize} />

The minimum usable size is **24px**. Below 24px the letterforms become illegible. The 16px size is reserved for the browser favicon only — never use it in the UI.

## Guidelines

<table>
  <thead>
    <tr><th>Do</th><th>Don't</th></tr>
  </thead>
  <tbody>
    <tr>
      <td>Use <code>--accent</code> for the letterforms and <code>--fg</code> for the dot</td>
      <td>Hardcode hex values — the logo will break in the opposite theme</td>
    </tr>
    <tr>
      <td>Use the <code>RJLogo</code> component — it handles tokens and props automatically</td>
      <td>Copy the raw SVG and apply color attributes directly</td>
    </tr>
    <tr>
      <td>Respect the minimum size of 24px in the UI</td>
      <td>Scale below 24px outside of a favicon context</td>
    </tr>
    <tr>
      <td>Place the logo on <code>--bg</code> or <code>--surface</code> backgrounds for full contrast</td>
      <td>Place the logo on <code>--accent</code> — the burgundy letterforms will disappear</td>
    </tr>
  </tbody>
</table>

## Component

Import `RJLogo` from the `ui` directory. Pass a `size` prop to control the rendered dimensions. Colors are applied via CSS tokens automatically.

```jsx
import RJLogo from '../components/ui/RJLogo'

// Default — 40px, tokens applied from CSS
<RJLogo size={40} />

// Explicit size
<RJLogo size={56} />
```

## SVG structure

The SVG has three paths. The first two form the R and J letterforms; the third is the dot. All three accept a `fill` attribute or inherit from CSS.

```jsx
<svg width={size} height={size * (41 / 36)} viewBox="0 -2 36 41" fill="none">
  {/* Left stem (R) */}
  <path d="M0.5 33.5V3.5…" fill="var(--accent)" />
  {/* Right bowl + J arc */}
  <path d="M29.7754 2.56878…" fill="var(--accent)" />
  {/* Dot */}
  <path d="M35.5 32C35.5 34.76…" fill="var(--fg)" />
</svg>
```

## With Framer Motion

The navbar wraps `RJLogo` in a `motion.div` to add hover and click interactions.

```jsx
import { motion } from 'framer-motion'
import RJLogo from '../components/ui/RJLogo'

const SPRING = { type: 'spring', stiffness: 400, damping: 17 }

<motion.div
  whileHover={{ scale: 1.1 }}
  whileTap={{ rotate: 180, scale: 0.5 }}
  transition={SPRING}
  style={{ display: 'block', cursor: 'pointer' }}
>
  <RJLogo size={56} />
</motion.div>
```
